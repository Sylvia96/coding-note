## Common data structures
* Array & String（数组与字符串）

*Convert a string to an array, and operate on **each character** in the string*

**Pros:**

*simple*

*O(1): Query by index*

**Cons:**

*continuous space*

*O(n): (Query if a[n] exists -> traverse the whole array) / (Add/delete a[n])*

* Linked list（链表）
* Stack（栈）
* Queue（队列）
* Deque（双端队列）
* Tree（树）
